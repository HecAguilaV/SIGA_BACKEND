# Manual de Usuario - Ecosistema SIGA

## Bienvenida
Gracias por elegir SIGA. Este manual le guiará a través de las funcionalidades de nuestra plataforma integral.

---

## 1. Web Comercial (Portal de Clientes)
**Acceso**: [https://siga-web.vercel.app](https://siga-web.vercel.app)

### Registro y Suscripción
1. Navegue a la sección "Planes".
2. Seleccione el plan adecuado (Emprendedor, PYME, Corporativo).
3. Complete el formulario de registro.
4. Recibirá sus credenciales de administrador vía correo.

---

## 2. Web App (Sistema de Gestión)
**Acceso**: [https://siga-appweb.vercel.app](https://siga-appweb.vercel.app)

### Inicio de Sesión
- Utilice el correo y contraseña registrados.
- El sistema detectará su rol automáticamente.

### Dashboard Principal
- **Resumen**: Visualice ventas del día y stock crítico.
- **Asistente IA**: Haga clic en el icono flotante de SIGA para hablar o escribir comandos (ej: "Crear producto Bebida a 1500").

### Gestión de Inventario
- Menú Lateral > Inventario.
- Use los filtros para buscar por local o categoría.
- Acciones: Editar (lápiz) o Eliminar (papelera).

---

## 3. App Móvil (Operaciones en Terreno)
**Descarga**: Disponible en Play Store (Beta) o vía APK directo.

### Funcionalidades
- **Login Biométrico**: Si su dispositivo lo soporta, active el ingreso con huella.
- **Consulta Rápida**: Escanee o busque productos para ver stock en tiempo real.
- **Ajustes de Stock**: Seleccione un producto y actualice la cantidad disponible en la estantería.

---

## 4. Soporte y Ayuda
Si requiere asistencia técnica:
- **Asistente Virtual**: Pregunte directamente a la IA de SIGA.
- **Contacto**: help@siga.cl
