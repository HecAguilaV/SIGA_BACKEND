# Documentación SIGA Backend (Diciembre 2025) 

Este directorio contiene la documentación profesional, consolidada y definitiva del sistema SIGA Backend.

## Índice

- `API_ENDPOINTS.md` – Especificación exhaustiva de todas las rutas públicas del backend.
- `ENDPOINTS_OPERATIVOS.md` – Estado de verificación y uso por cada cliente oficial.
- `ANALISIS_ENDPOINTS_Y_SINCRONIZACION.md` – Flujos de sincronización App/Web y acuerdos inter-equipos.
- `ARQUITECTURA_BACKEND.md` – Fuente de verdad de módulos backend y decisiones de diseño.
- `ESQUEMAS_DATABASE.md` – Modelo relacional completo de `siga_saas` y `siga_comercial`.
- `SINCRONIZACION_APPMOVIL_WEBAPP.md` – Guía práctica de consumo simultáneo WebApp ↔ App Móvil.
- `VISION_SIGA_ASISTENTE_IA.md` – Hoja de ruta y capacidades del asistente inteligente.
- `ECOSISTEMA/` – Especificaciones técnicas de WebApp, Portal Comercial y App Móvil.

Todos los documentos se mantienen sincronizados al cierre de iteraciones mayores. Para cambios sustantivos, actualizar el archivo correspondiente y registrar el enlace en este índice.
